# Dolar Info

Queres saber cuanto esta el Dolar Oficial y el Dolar blue pero no queres entrar a una pagina web? Esta es la tool de consola para vos :).

## Instalacion

````bash
npm install -g dol-info
````

## Correrlo

````bash
dol-info
````
